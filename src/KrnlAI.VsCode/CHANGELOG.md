# Changelog

## 1.0.0 (2026-06-15)

### 🎉 Initial Marketplace Release

- **Chat** with full editor context (open file, selection, diagnostics)
- **Inline Code Completion** with LRU-cached responses
- **21 Slash Commands**: /explain, /fix, /test, /refactor, /review, /doc, /commit, /run, /build, /task, and more
- **Cognitive Dashboard**: cycle stages, risk levels, safety status, emotional state
- **7 Memory Types** explorer: Semantic, Episodic, Procedural, Working, Emotional, Autobiographical, Temporal
- **Policy Viewer**: list, filter, view active policies and experiments
- **Safety System (R01-R20)**: multi-layer governance with allowlist, risk scoring, metacritic
- **Emotional State (VAD)**: mood indicator in status bar + dashboard
- **3 Approval Modes**: Chat (safe), SafeAgent (auto-review), FullAgent (full governance)
- **CodeLens**: Explain / Test / Review lenses on functions and classes
- **Code Actions**: Fix diagnostics, Explain selection, Generate Test
- **Hover Provider**: on-demand code explanations
- **Terminal Integration**: run commands with allowlist/blocklist
- **Git Integration**: commit, diff, branch, status, log, PR review
- **Agentic Loops**: multi-step autonomous task execution
- **13+ LLM Providers**: OpenAI, Anthropic, Google, DeepSeek, Ollama, OpenRouter, and more
- **Multi-platform**: Windows, macOS, Linux
- **Runtime Modes**: embedded (Sidecar), local API, remote API
